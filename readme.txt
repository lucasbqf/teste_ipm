Ola! me chamo lucas Baldessar e esse projeto foi criado para o teste da vaga  Desenvolvedor na IPM-sistemas


Segundo os requisitos para realização do teste, nao foi informado nenhum framework ou guideline para a execução do projeto
sendo assim, eu acabei usar apenas o PHP puro sem adição de nenhum framework (alem do bootstrap para o front) para a realização do teste.

o sistema esta dividido em 2 o backend fica na pasta "api" e o frontend fica na pasta "Taskmngr"
no projeto a apenas um endpoint que realiza o CRUD das tarefas se chamaando Task.php.

o backend foi dividido em 3 partes e seguindo conceito de MVC.

o frontend esta basicamente usando a Ideia de templates onde os acessos são realizados nas paginas index.php,
altertask.php e newtask.php sendo os tamplates criados para facilitar a renderização das paginas.
na pasta handlers estão o objeto que realiza as requisições REST para o backend e tambem todas as Actions realizadas nos Forms

para o funcionamento do projeto ha uma pasta chamada config, onde fica as configurações do banco de dados e
tambem uma configuração para alterar facilmente os endereços de url das requisições( caso o projeto seja salvo e executado
em uma pasta diferente da Htdocs)

para a criação do banco de dados é necessario executar os comandos SQL que estão do arquivo "create_db.sql"
ja vindo com 2 tarefas para teste.
